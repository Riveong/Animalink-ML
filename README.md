# ML Model Deployment Simplified by me  
Requirements:  
- Python3  
- Tensorflow 2.15+  
- pip 23.3+  
   
   ## How to use  
   1. Install the requirements libraries
   ```py
    python3 -m pip install -r requirements.txt
   ```  
   2. ensure that uvicorn is already installed, and start the uvicorn server  
   ```py
   python3 -m uvicorn main:app --reload
   ```
